Amadeus-Neo Privacy Policy

Last updated: June 14th, 2026

Contact:

  • Discord: @ttl0

Amadeus-Neo is a Discord bot used for moderation, verification, anti-spam, activity roles, and server boost perks.

Summary for Normal People

We do not:
  • Collect or stores any more data than required to function.
  • Sell user data.
  • Share user data.
  • Train AI Models with user data.

Who Controls the Data

For data stored by Amadeus-Neo, the bot operator controls how that data is used. Discord separately controls data processed by the Discord platform. Server owners and admins control how the bot is configured in their servers.

Data We Process

The bot may process limited Discord data, including:

Data Type Purpose
User IDs Used to identify users for verification, moderation, activity roles, boost perks, and privacy requests.
Usernames and display names Used to show readable names in moderation logs, alerts, and admin review messages.
Server, channel, role, message, and emoji IDs Used to apply server settings, manage roles and emojis, monitor configured channels, and reference moderation actions.
Timestamps Used to record when bot actions, moderation events, verification events, or activity events happen.
Bot configuration Used to save server settings so enabled modules continue working after restarts.
Verification status Used to check whether users have completed server verification.
Activity counts Used to grant configured activity roles and detect abuse of activity systems.
Moderation actions and security flags Used to prevent abuse, detect spam or compromised accounts, and help moderators review bot actions.
Boost perk settings Used to manage user-requested boost perks, such as role names, colors, emoji names, or images.

Message Content

Amadeus-Neo may check message content in real time for moderation, verification, anti-spam, honeypot, activity, or user-requested features.

Amadeus-Neo does not store normal message history.

Message content may be checked to detect spam, scams, suspicious links, compromised accounts, honeypot triggers, CAPTCHA responses, or abuse. After processing, the message body is not retained unless a user voluntarily sends information needed for a feature, such as a boost perk role name, emoji name, color, or image.

How We Use Data

We use data only to:

  • operate the bot;
  • apply server settings;
  • verify users;
  • detect spam, scams, or abuse;
  • perform moderation actions;
  • manage activity roles and boost perks;
  • debug problems;
  • protect the bot and Discord servers;
  • respond to abuse, safety, or privacy requests.

Legal Basis for European and UK Users

If GDPR, UK GDPR, or similar privacy law applies, we process data under the following legal bases:

  • Legitimate interests: to operate the bot, prevent spam, detect abuse, protect Discord servers, apply moderation actions, and keep the bot secure.
  • Requested service: to provide features requested by users or server admins, such as verification, activity roles, and boost perks.
  • Legal obligation: when we must keep, disclose, or delete information to comply with applicable law.

Where Data Is Stored

Amadeus-Neo stores its data in a self-contained database controlled by the bot operator. It does not use external analytics, advertising, tracking, or AI services.

Provider Location Purpose
OVH Beauharnois (BHS), Canada Hosts the server where Amadeus-Neo and its self-contained database run.

Because the bot is hosted in Canada, bot data may be stored outside your country, including outside the European Economic Area or United Kingdom.

Who Can See Data

Data may be visible to:

  • the bot operator;
  • trusted maintainers;
  • server admins or moderators, when the bot sends logs or alerts;
  • Discord, because the bot runs on Discord;
  • the hosting provider used to run the bot;
  • legal or safety parties if required by law or needed to prevent abuse.

Data Retention

Data is kept only as long as needed for the bot's features, safety, debugging, moderation, abuse prevention, or legal reasons.

Server configuration is kept while the bot is used in a server. Activity counts, verification records, moderation records, security flags, and boost perk records are kept only as long as needed to provide those features. Temporary data may be deleted automatically when it expires or is no longer needed.

Your Rights

Depending on where you live, you may have the right to request access to, correction of, deletion of, restriction of, objection to, or a copy of your personal data.

You can make a request about bot-controlled data linked to your Discord user ID by contacting us at:

  • Discord: @ttl0

Please include your Discord user ID and the server involved. We may need to verify that you control the account before acting on a request.

Some data may not be deleted immediately if it is needed for security, abuse prevention, moderation records, backups, or legal reasons.

Children

Amadeus-Neo is not intended for children under the minimum age required to use Discord.

Security

We use reasonable safeguards to protect bot data, including limiting access to the bot database and storing only the data needed for the bot to work. No system is perfectly secure.

Changes

We may update this Privacy Policy when the bot changes. The latest version will be posted on this page.
